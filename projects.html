Estimat Runoff

The SCS curve number method is a simple, widely used and efficient method for determining the approximate amount of runoff from rainfall. The web application we developed is based on the SCS-CN method only and depending on data given by the user it can estimate total runoff depth, runoff volume for a given area, and more insights.
